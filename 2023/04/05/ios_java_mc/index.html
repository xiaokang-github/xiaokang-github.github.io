在苹果手机上玩 Minecraft JE

Xiao Kang
2023-04-05

简介

PojavaLuancher 可以在手机上玩 Minecraft JE

要求

iOS 12.2 或更高版本
iPhone 6s 或更高

安装

这里要使用 TrollStore
可以看看 3o 米的 教程 （懒得写）
先在 Github 仓库下载签名文件
下载完的文件点分享
然后在文件分享界面选择 TroolStore
按提示操作
然后你就可以玩了
就这么简单

帮助

掉签了打开 TrollStore 续签就行
